# 🎨 Maquette Art — Blog Street Art

Une maquette de blog dédié au **street art**, réalisée en **HTML / CSS**, avec un design mobile-first et une identité visuelle forte.

---

## 📋 Présentation

**Maquette Art** est le blog d'Alvin, passionné de street art. Le site présente les actualités du monde du street art, des interviews d'artistes, une galerie et une newsletter. La maquette est conçue pour une lecture sur mobile avec une typographie soignée et une charte graphique cohérente.

---

## ✨ Fonctionnalités

- 🖼️ **Slider** : bannière d'accueil animée avec texte centré (bxSlider / jQuery)
- 📰 **Section Actualités** : articles de presse avec image, date et bouton "Lire la suite"
- 🎤 **Section Interviews** : portraits d'artistes avec résumé et lien
- 💬 **Citation mise en avant** : bloc citation stylisé (Banksy)
- 📧 **Newsletter** : formulaire d'inscription dans le footer
- 📱 **Réseaux sociaux** : icônes Twitter, Facebook, Instagram (Font Awesome)
- 📐 **Design mobile-first** : mise en page pensée pour les petits écrans

## ⚙️ Installation

Aucune dépendance locale — toutes les librairies sont chargées via CDN.

**1. Cloner le dépôt :**

```bash
git clone https://github.com/votre-utilisateur/maquette-art.git
```

**2. Ouvrir dans le navigateur :**

```
index.html
```

> Aucun serveur local requis — le fichier peut être ouvert directement.
